# Proyecto-A-b-I

Un pequeño rincón en la web hecho con muchísimo cariño **solo para Susan Abigail Lacayo** 💚🌸💙

Una página animada, en colores pastel (verde suave, rosado suave y azulitos cálidos),
que celebra las cosas que ella ama: **Soda Stereo, Morat, Michael Jackson, Adele,
las fresas** 🍓 y su chihuahua café **Inu** 🐾. Incluye sus fotos, animaciones suaves
y un mensaje dedicado a lo capaz, inteligente y luminosa que es.

## 🛠️ Hecho con
- **HTML5** — estructura
- **CSS3** — animaciones, keyframes y diseño pastel responsivo
- **JavaScript (vanilla)** — partículas flotantes, reveal al hacer scroll y corazones interactivos

Sin dependencias ni frameworks: se abre directo en el navegador.

## 📁 Estructura
```
Abi/
├── index.html                # la página
├── assets/
│   ├── css/
│   │   ├── styles.css         # estilos y paleta pastel
│   │   └── animations.css     # keyframes y micro-animaciones
│   ├── js/
│   │   └── main.js            # partículas e interacciones
│   └── img/
│       ├── susan-bebe.jpg     # (agregar) foto de pequeña
│       └── susan-nina.jpg     # (agregar) foto de niña
└── README.md
```

## ▶️ Cómo verla
Solo abre `index.html` en cualquier navegador. Para que se muestren las fotos,
coloca las imágenes en `assets/img/` con los nombres indicados (ver `assets/img/LEEME.txt`).
